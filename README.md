# Professional Student Portal

This is a **fully responsive student portal** project built with **HTML, CSS, and JavaScript**.

## Features
- Login with email and password
- Profile section showing **name, age, and email**
- Dashboard showing **9 courses with grades**
- Add, edit, and delete courses
- Fully responsive design
- Optional background image for professional look

## Usage
1. Clone or download the repository
2. Open `index.html` in a browser (or on your phone using a code editor like Acode)
3. Login or create an account to access the dashboard
